1.先配置config.txt文件
       user_name = 'asf5668ad'     // ev商户的用户名
       pwd = 'fa66e1da1'     // ev商户的密码
       time_gap = 5   // 每多少秒重新拉取数据监测
       receiver = '8546564@qq.com'  // 用户接收告警的邮箱
       continues_play_gap = 60  // 连续播放视频之间的时间间隔，如果两个视频之间的播放时间间隔小于该时间，将视为翻录
       continues_play_max = 5   // 连续播放的视频次数如果大于该数目，将会发出邮件报警
       play_max_day = 20  // 一天内最大播放次数
       size = 10  // 监视前多少个激活码
       recall = True  // 如果一天内播放次数超过最大播放次数，是否召回激活码，如果不召回，改成False

2.在同级目录下新建目录名为Logs的文件夹，与config.txt、EV防翻录检测.exe放在一起

3.运行EV防翻录检测.exe


注：1.该程序会在后台运行，占用内存非常小。白天或者晚上可以打开运行挂在后台；
       2.配置文件请安格式顺序书写，所填写的数字不要小于等于0；
       3.该程序会对已经激活了的用户进行监控，详细的每节播放时间请在Logs文件夹中找到对应激活码的文件打开查看；
       格式是：时间-剩余播放次数-对应id
       4.程序会根据配置文件设置的周期时间定期去查看新数据，如果有用户激活了激活码，会被加入到监控名单中进行监控；
       5.如果邮箱填写正确，若发生用户行为异常进行告警，邮件可能会发到垃圾邮箱中(比如qq邮箱)。
